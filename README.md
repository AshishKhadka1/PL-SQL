# Oracle SQL & PL/SQL Learning Repository

A structured collection of Oracle SQL and PL/SQL scripts designed for learning, practicing, and revising database concepts using Oracle SQL Developer. The repository progresses from SQL fundamentals to advanced PL/SQL programming through practical examples.

---

## Learning Path

### 01_SQL_Basics.sql
Fundamental SQL concepts and database operations.

**Topics Covered**
- Database Objects
- CREATE TABLE
- INSERT
- SELECT
- WHERE Clause
- ORDER BY
- DISTINCT
- LIKE
- BETWEEN
- IN / NOT IN
- AND / OR / NOT
- UPDATE
- DELETE
- COMMIT
- ROLLBACK
- SAVEPOINT

---

### 02_SQL_Intermediate.sql
SQL functions and data analysis techniques.

**Topics Covered**
- Aggregate Functions
  - COUNT()
  - SUM()
  - AVG()
  - MAX()
  - MIN()
- GROUP BY
- HAVING
- Column Alias
- Arithmetic Expressions
- ROWNUM
- SYSDATE
- Character Functions
- Numeric Functions
- Conversion Functions

---

### 03_SQL_Advanced.sql
Advanced SQL concepts and Oracle database objects.

**Topics Covered**
- Views
- Sequences
- Constraints
- Set Operators
- Joins
- Referential Integrity
- Subqueries
- Data Dictionary
- Reports
- Input Variables
- DEFINE Variables
- Tablespaces
- Users and Roles

---

### 04_PL_SQL_Fundamentals.sql
Introduction to PL/SQL programming.

**Topics Covered**
- PL/SQL Blocks
- Variables
- %TYPE
- IF / ELSIF / ELSE
- LOOP
- WHILE LOOP
- FOR LOOP
- Nested Loops
- Implicit Cursor
- Explicit Cursor

---

### 05_PL_SQL_Advanced.sql
Advanced PL/SQL programming concepts.

**Topics Covered**
- Triggers
- Stored Procedures
- Functions
- Exception Handling

- Packages
